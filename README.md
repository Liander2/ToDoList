# To Do List

A List generating app where the user can add items to the list and remove said items when they are done or remove the entire list at the same time.
Widgets used in this project.

- Scaffold
- AppBar
- Padding
- TextWidgets
- Column
- FloatingActionButton
- TextButton
- Color
- Container
- Checkbox
- Expanded
